INTRODUCTION
------------

A novel architecture to reduce delays and instantly alarm the drowsy drivers using deep learning object detection technologies.

-------
CONTENT
-------
1. Data
	a.Train
	b.Test
2. Models
	a.cnnCat2.h5 - open and close eye detection
	b.yawnCNN.h5 - yawn and no yawn	detection
3. Haar cascade files
4. alarm.wav
5. CNNModelTraining.ipynb - CNN model training jupyter notebook
6. VGG16ModelTraining.ipynb - VGG16 model training jupyter notebook
7. drowsiness detection.py - Real time detection code

------------
Dependencies
------------
1. Open CV2
2. Keras
3. Numpy
4. pygame

------------------------------------------------
Requirements to run the drowsiness detection.py
------------------------------------------------
1. Download and maintain the same folder structure
2. Install all dependencies
3. Run the command python3  drowsiness detection.py
4. Allow access to open computers camera
